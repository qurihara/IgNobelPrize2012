Speech Jammer (Ig Nobel Prize 2012 Acoustics Prize) with HTML5

This Simulator simulates 'SpeechJammer', which is the Acoustics Prize in Ig Nobel Prize 2012, using Web Audio API and Web RTC(getUserMeida()). Here is an official explanation of the prize. Visit here to see for more Ig Nobel Prize 2012 informations. Kazutaka Kurihara, who created the concept of SpeechJammer and build the device.
